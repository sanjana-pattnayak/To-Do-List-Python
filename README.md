# 📝 To-Do List Application (CLI)

## 📌 Description
A simple console-based To-Do List app built using Python.  
It allows users to add, view, and remove tasks.  
Tasks are stored in a file for persistence.

## 🚀 Features
- Add tasks
- View tasks
- Remove tasks
- Data saved in text file

## 🛠️ Tech Used
- Python
- File Handling

## ▶️ How to Run
python todo.py
